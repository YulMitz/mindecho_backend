## POST api/auth/register
Request Body:
```
{
    "email": "123456789@gmail.com"
    "password": "8888888" // 最少6個字元
    "firstName": "Yuming"
    "lastName": "Mitzgo"
    "dateOfBirth": "2003-09-21"
}
```
Return:
```
{
    message: 'User registered successfully',
    token,
    user: {
    id: user._id,
    email: user.email,
    firstName: user.firstName,
    lastName: user.lastName
    }
}
```
